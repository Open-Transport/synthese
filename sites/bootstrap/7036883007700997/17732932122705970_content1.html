Warning This version does not contains the glyph icons.

Use

To use bootstrap in your CMS application, update your template acccording to the following instructions.

HTML doctype

Put this code at the beginning of the content :

<!DOCTYPE html>

Header

Put this code somewhere in the <head> section :

<link href="/bootstrap/css/bootstrap.min.css" rel="stylesheet"></link>
<link href="/bootstrap/css/bootstrap-responsive.min.css" rel="stylesheet"></link>

Footer

Put this code just before the </body> tag :

<script src="/bootstrap/js/bootstrap.min.js"></script>
